## Installation

1. Run `composer require flux-se/sylius-stripe-plugin`.

2. Import configuration
    ```yaml
    # config/packages/_sylius.yaml

    imports:
    # ...
    - { resource: "@FluxSESyliusStripePlugin/config/config.yaml" }
    ```
